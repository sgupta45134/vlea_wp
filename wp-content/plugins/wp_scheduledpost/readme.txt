== Description ==

RS Recurring Post Schedule Plugiin - Schedule a post to be published and unpublished by giving dates for same.

The plugin is i18n ready.

== Screenshots ==

Not currently available.

== Installation ==

Upload the RS Recurring Post Schedule Plugiin to your blog (in the 'wp-content/plugins' directory) (be sure to upload the "wp_scheduledpost" directory or it will not work.

Activate it in the plugins menu in your wp-admin control panel
